# Quick Recording Checklist

## Pre-Recording Setup

### Browser Setup:
- [ ] Clear browser cache/cookies or use incognito
- [ ] Set browser zoom to 100%
- [ ] Close unnecessary tabs
- [ ] Use Chrome or Firefox for best compatibility

### Screen Recording:
- [ ] Open screen recording software (OBS, Camtasia, etc.)
- [ ] Set resolution to 1920x1080 or higher
- [ ] Frame rate: 30fps minimum
- [ ] Test audio levels
- [ ] Test screen capture area
- [ ] Ensure cursor is visible and large enough

### Demo Account Setup:
- [ ] Create a test account (email/password) OR use Google
- [ ] Create 4-5 sample tasks with:
  - [ ] Different difficulty levels (easy, medium, hard)
  - [ ] Mix of tasks with and without dates
  - [ ] At least one overdue task
  - [ ] At least 2 completed tasks
  - [ ] At least 2 active tasks
- [ ] Complete at least 1 Pomodoro session for stats
- [ ] Earn some points from completed tasks

### Environment:
- [ ] Quiet recording space
- [ ] Good microphone connected
- [ ] Adequate lighting (if face cam)
- [ ] Close distracting apps/notifications
